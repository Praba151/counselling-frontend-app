# counselling Service

A full-stack online counseling platform that connects clients with licensed counselors through secure video calls, real-time chat, and appointment scheduling.


- Register and Login as Client or Counselor
- Counselor profile with bio, expertise, and available slots
- Appointment booking with session type selection
- Razorpay payment integration (INR)
- Daily.co video call per appointment
- Real-time chat using Socket.io
- Session notes with file attachments
- Protected routes based on user role
